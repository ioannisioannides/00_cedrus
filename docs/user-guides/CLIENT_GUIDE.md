# Client User Guide

## Overview

This guide explains how to use the Cedrus platform for clients.

## Key Features

- **Dashboard**: View certification status and upcoming audits.
- **Audit Prep**: Upload required documents.
- **Findings**: Respond to non-conformities.

## Workflows

1. **Document Submission**: Upload files before audit.
2. **Audit Review**: View audit plan and report.
3. **Corrective Actions**: Submit root cause analysis and evidence.
