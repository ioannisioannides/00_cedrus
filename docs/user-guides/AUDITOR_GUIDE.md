# Auditor User Guide

## Overview

This guide explains how to use the Cedrus platform for conducting audits.

## Key Features

- **Dashboard**: View assigned audits and tasks.
- **Audit Execution**: Complete checklists, raise findings, and upload evidence.
- **Reporting**: Generate audit reports.

## Workflows

1. **Audit Planning**: Review schedule and prepare plan.
2. **On-site Audit**: Execute audit steps.
3. **Reporting**: Draft and submit report.
